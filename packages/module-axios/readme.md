# module-axios

基于 `axios` 的模块化接口请求、管理解决方案，轻松管理项目中接口。

## 功能

- 模块化接口管理
- 多环境、多结构动态支持

## 安装

Using npm:

```bash
npm install module-axios
```

Using yarn:

```bash
yarn add module-axios
```
